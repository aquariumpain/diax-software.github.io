Commands

Command Name Aliases Arguments Description
credits Display the credits for Diax.
help Displays the help message for Diax.
ping pang, peng, pong, pung Shows the message response time and api ping.
report description of the bug Used for reporting bugs to Diax's developers.
suggest description of the suggestion Used to suggest features for Diax.
join Summons Diax to your voice channel.
nowplaying song, current, nowplaying, np Shows the currently playing song.
play YouTube URL, direct link to song, or a search query. Plays a track from a URL or query.
queue Displays the songs in the queue.
repeat Toggles the repeating function.
shuffle Shuffles the queue.
skip Skips the current track.
stop Stops the player.
volume Amount 0-150 (Default 100) Sets the volume of Diax.
hug @mention the person(s) to hug. Hugs the mentioned users.
